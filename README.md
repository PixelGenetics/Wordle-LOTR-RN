Hola!

Para instalar el proyecto en tu dispositivo Android, primero debes descargar el APK. Puedes hacerlo desde el siguiente enlace:

https://drive.google.com/file/d/1xuu6DXFEh9YqARt_4hQ7-Jk57GyoqU7Y/view?usp=drive_link 

Una vez que abras el instalador, verás la siguiente ventana:

![image alt](https://github.com/PixelGenetics/Wordle-LOTR-RN/blob/main/assets/images/1.jpg?raw=true)

De forma predeterminada, Android recomienda realizar un análisis de seguridad de la aplicación. Si deseas omitir este paso, presiona el botón More details y luego selecciona Install without scanning.

![image alt](https://github.com/PixelGenetics/Wordle-LOTR-RN/blob/main/assets/images/2.jpg?raw=true)

Después de esto, comenzará la instalación.

Si prefieres que Android analice la aplicación antes de instalarla, el proceso puede tardar varios minutos y verás una pantalla similar a esta:

![image alt](https://github.com/PixelGenetics/Wordle-LOTR-RN/blob/main/assets/images/3.jpg?raw=true)

Cuando finalice la instalación, la ventana se verá así:

![image alt](https://github.com/PixelGenetics/Wordle-LOTR-RN/blob/main/assets/images/4.jpg?raw=true)

Presiona el botón Open para iniciar el juego.

![image alt](https://github.com/PixelGenetics/Wordle-LOTR-RN/blob/main/assets/images/5.jpg?raw=true)

¡Y listo!

Gracias por probar mi aplicación. 😊
